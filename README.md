# 🏍️ Honrados do Asfalto — Site Oficial

Site oficial do moto grupo **Honrados do Asfalto**, construído com React, TypeScript e Tailwind CSS. Um site moderno, responsivo e inspirado na cultura biker.

## 📌 Sobre o Projeto

Este projeto consiste na criação de um site completo, porém simples, com design moderno e foco na experiência do usuário, inspirado na cultura biker e nos valores de liberdade, irmandade e respeito.

O site será totalmente responsivo, garantindo uma navegação fluida em dispositivos móveis, tablets e desktops.

---

## 📋 Seções

| Seção | Descrição |
|-------|-----------|
| **Home** | Banner hero com imagem de fundo, nome do grupo e slogan |
| **Sobre Nós** | História do grupo e valores: Honra, Paixão e Irmandade |
| **Membros** | Apresentação dos integrantes com fotos e cargos |
| **Eventos** | Agenda de encontros, passeios e eventos futuros |
| **Galeria** | Fotos do grupo com integração ao Instagram (Elfsight) |
| **Contato** | Formulário de contato, redes sociais e informações |

## ✨ Funcionalidades

- ✅ Design responsivo (mobile, tablet e desktop)
- ✅ Animações suaves com Framer Motion
- ✅ Navegação intuitiva com scroll suave
- ✅ Botão fixo de WhatsApp
- ✅ Integração com feed do Instagram via Elfsight
- ✅ Paleta de cores preto e branco com identidade biker
- ✅ Tipografia marcante e estilizada

## 🛠️ Stack Tecnológica

- **React 18** — Biblioteca de UI
- **TypeScript** — Tipagem estática
- **Vite** — Build tool ultrarrápido
- **Tailwind CSS** — Estilização utilitária
- **shadcn/ui** — Componentes acessíveis
- **Framer Motion** — Animações
- **Lucide React** — Ícones
- **React Router** — Roteamento SPA

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/SEU-USUARIO/mg_honrados_do_asfalto.git
cd mg_honrados_do_asfalto

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O site estará disponível em `http://localhost:8080`.

## 📦 Scripts disponíveis

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção |
| `npm run preview` | Pré-visualiza o build de produção |
| `npm run lint` | Executa o linter |
| `npm run test` | Executa os testes |
